* [ ] Gerekli paketleri kurmak için aşağıdaki kodu çalıştır:

```r
install.packages("wbstats")
install.packages("dplyr")

```

* [ ] Gerekli paketleri aktif etmek için aşağıdaki kodu çalıştır:

```r
library(wbstats)
library(dplyr)

```


> Not: OECD verileri bu projede kullanılmıyor; çünkü `OECD` paketinin `get_dataset()` ve `search_dataset()` fonksiyonları, OECD eski SDMX API altyapısını kapattığı için artık çalışmıyor (her sorgu "HTTP 404" hatası veriyor). 
> 
> Bunun yerine aynı makro göstergeler, sorunsuz çalışan Dünya Bankası (`wbstats`) üzerinden çekiliyor. 

* [ ] İstediğin göstergenin Dünya Bankası kodunu bulmak için arama yap (örnek: işsizlik); arama açıklamayla yapılır:

```r
gostergeler <- wb_search("unemployment, total")
head(gostergeler)

```

* [ ] Türkiye işsizlik verisini çek:

```r
issizlik <- wb_data(country = "TR", indicator = "SL.UEM.TOTL.ZS")

```

* [ ] Türkiye enflasyon (TÜFE yıllık %) verisini çek:

```r
enflasyon_wb <- wb_data(country = "TR", indicator = "FP.CPI.TOTL.ZG")

```

* [ ] Türkiye GSYH büyüme (%) verisini çek:

```r
buyume <- wb_data(country = "TR", indicator = "NY.GDP.MKTP.KD.ZG")

```

* [ ] Her veri setinin doğru geldiğini ilk satırları görüntüleyerek kontrol et:

```r
head(issizlik)
head(enflasyon_wb)
head(buyume)

```
