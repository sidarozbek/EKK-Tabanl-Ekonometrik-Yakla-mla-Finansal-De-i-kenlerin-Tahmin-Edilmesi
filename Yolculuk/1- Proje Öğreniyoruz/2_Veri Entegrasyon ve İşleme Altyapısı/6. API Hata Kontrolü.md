* [ ] RStudio'yu aç ve proje dosyanı yükle
* [ ] Gerekli paketleri kurmak için aşağıdaki kodu çalıştır:

```r
install.packages("httr")

```

* [ ] Gerekli paketi yükle ve test edeceğin API adresini `url` değişkenine ata (yoksa `GET` çağrısı hata verir):

```r
library(httr)
url <- "https://evds3.tcmb.gov.tr/igmevdsms-dis/series=TP.TUKFIY2025.GENEL&startDate=01-01-2015&endDate=01-01-2025&type=json"

```

* [ ] EVDS anahtarını ortam değişkenine kaydet (kendi anahtarını yapıştır; bu satırı dosyaya kaydetmeyebilirsin, sadece Console'da bir kez çalıştırman yeterli):

```r
Sys.setenv(EVDS_API_KEY = "KENDI_EVDS_ANAHTARIN")

```

* [ ] API bağlantısını test etmek için isteği gönder; anahtar header ile gönderilir ve okurken `Sys.setenv`'de kullandığın adın aynısını (`EVDS_API_KEY`) yazman gerekir:

```r
response <- GET(url, add_headers(key = Sys.getenv("EVDS_API_KEY")))

```

* [ ] API bağlantı durumunu kontrol etmek için aşağıdaki kodu çalıştır:

```r
status_code(response)

```

* [ ] Console ekranında sonuç olarak `200` değerinin geldiğini doğrula
* [ ] API'den veri dönüşünü kontrol et ve gelen içeriğin gerçekten JSON olduğunu doğrula; çıktının başında `{"totalCount"` görmelisin, `<!DOCTYPE html>` veya `401`/`403` görüyorsan adres ya da anahtar hatalıdır:

```r
content(response, "text")

```

* [ ] Hatalı API anahtarı girerek sistemin hata verdiğini test et; yanlış anahtarda sistem `401` (yetkisiz) döndürmeli:

```r
response_hatali <- GET(url, add_headers(key = "YANLIS_ANAHTAR"))
status_code(response_hatali)

```

* [ ] İnternet bağlantısı kapalıyken API isteğinin başarısız olduğunu doğrula
* [ ] Hem internet kopmasını hem de 200 dışındaki HTTP yanıtlarını (401, 403, 400, 404 gibi) yakalayan hata yönetimi kodunu yaz:

```r
tryCatch({
  response <- GET(url, add_headers(key = Sys.getenv("EVDS_API_KEY")))
  if (status_code(response) != 200) {
    stop(paste("API hata kodu:", status_code(response)))
  }
  print("Bağlantı başarılı")
}, error = function(e) {
  print(paste("API bağlantı hatası:", e$message))
})

```

* [ ] Console ekranında doğru mesajın görüntülendiğini doğrula; geçerli anahtarla "Bağlantı başarılı" yazmalı
