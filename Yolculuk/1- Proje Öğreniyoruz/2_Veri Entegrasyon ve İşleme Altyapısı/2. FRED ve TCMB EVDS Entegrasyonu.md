* [ ] [TCMB EVDS3](https://evds3.tcmb.gov.tr/) sitesine gir ve hesabınla giriş yap
* [ ] Üst menüden "Başlıca Göstergeler" bölümüne tıkla
* [ ] Başlıca Göstergelerden "Tüketici Fiyat Endeksi (Enflasyon - TÜİK)" yazan ve çıkan sonuçlardan aylık TÜFE serisini seç
* [ ] Seçtiğin serinin kodunu (örneğin `TP_TUKFIY2025_GENEL-1`) not et; bu kod enflasyon verisinin seri kodudur
* [ ] Arama kutusuna "TCMB Ağırlıklı Ortalama Fonlama Maliyeti" yaz ve TCMB politika/fonlama faizi serisini seç
* [ ] Seçtiğin faiz serisinin kodunu (örneğin `TP_APIFON4`) not et; bu kod faiz verisinin seri kodudur
* [ ] Sağ üstte kullanıcı menüsünden "API Anahtarı" bölümüne gir ve kişisel anahtarını kopyala
* [ ] Anahtarını bir değişkene atamak için aşağıdaki kodu çalıştır ve tırnak içine kendi anahtarını yapıştır:

```r
api_key <- "EVDS_API_ANAHTARINI_BURAYA_YAZ"

```

* [ ] Enflasyon serisinin kodunu bir değişkene yaz:

```r
enflasyon_kodu <- "TP_TUKFIY2025_GENEL-1"

```

* [ ] Faiz serisinin kodunu bir değişkene yaz:

```r
faiz_kodu <- "TP_APIFON4"

```

* [ ] Seri kodlarını API formatına çevir (alt çizgileri noktaya çevirir, sondaki "-1" ekini kaldırır):

```r
enflasyon_api_kodu <- gsub("-1$", "", gsub("_", ".", enflasyon_kodu))
faiz_api_kodu <- gsub("-1$", "", gsub("_", ".", faiz_kodu))

```

* [ ] Enflasyon verisini çek (anahtar header'da gönderilir):

```r
enflasyon_verisi <- GET(
  paste0("https://evds3.tcmb.gov.tr/igmevdsms-dis/series=", enflasyon_api_kodu,
         "&startDate=01-01-2015&endDate=01-01-2025&type=json"),
  add_headers(key = api_key)
)

```

* [ ] Faiz verisini çek:

```r
faiz_verisi <- GET(
  paste0("https://evds3.tcmb.gov.tr/igmevdsms-dis/series=", faiz_api_kodu,
         "&startDate=01-01-2015&endDate=01-01-2025&type=json"),
  add_headers(key = api_key)
)

```

* [ ] Yanıtların başarılı olduğunu kontrol et; ikisinin de 200 döndüğünü doğrula:

```r
status_code(enflasyon_verisi)
status_code(faiz_verisi)

```

* [ ] Enflasyon verisini tabloya çevir ve temizle (aylık veri, tarih "yıl-ay" formatında gelir):

```r
enflasyon_tablo <- fromJSON(content(enflasyon_verisi, as = "text", encoding = "UTF-8"))$items %>%
  transmute(
    tarih = as.Date(paste0(Tarih, "-01"), format = "%Y-%m-%d"),
    deger = as.numeric(TP_TUKFIY2025_GENEL)
  )

```

* [ ] Faiz verisini tabloya çevir ve temizle (günlük veri olarak kalır, tarih "gün-ay-yıl" formatında okunur):

```r
faiz_tablo <- fromJSON(content(faiz_verisi, as = "text", encoding = "UTF-8"))$items %>%
  transmute(
    tarih = as.Date(Tarih, format = "%d-%m-%Y"),
    deger = as.numeric(TP_APIFON4)
  ) %>%
  filter(!is.na(deger))

```
