* [ ] EVDS'den veri çekmek için yardımcı fonksiyonu tanımla (her EVDS göstergesinde tekrar kullanılır; sen sadece seri kodunu vereceksin):

```r
evds_cek <- function(seri_kodu, baslangic = "01-01-2015", bitis = "01-01-2025") {
  url <- paste0("https://evds3.tcmb.gov.tr/igmevdsms-dis/series=", seri_kodu,
                "&startDate=", baslangic, "&endDate=", bitis, "&type=json")
  yanit <- GET(url, add_headers(key = Sys.getenv("EVDS_API_KEY")))
  fromJSON(content(yanit, as = "text", encoding = "UTF-8"))$items
}

```
