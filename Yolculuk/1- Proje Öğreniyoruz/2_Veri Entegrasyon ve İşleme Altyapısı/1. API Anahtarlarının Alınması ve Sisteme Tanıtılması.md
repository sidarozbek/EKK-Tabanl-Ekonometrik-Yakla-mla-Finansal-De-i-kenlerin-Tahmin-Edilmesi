* [ ] FRED resmî sitesi olan [FRED (Federal Reserve Economic Data)](https://fred.stlouisfed.org/?utm_source=chatgpt.com) adresine gir, ücretsiz hesap oluştur ve API key al
* [ ] TCMB EVDS sistemi olan [TCMB EVDS](https://evds3.tcmb.gov.tr/) sitesine gir, kayıt ol ve API anahtarını kopyala
* [ ] RStudio'daki kod sayfana dön ve gerekli paketleri kurmak için aşağıdaki kodu çalıştır:

```r
install.packages("fredr")

```

* [ ] Gerekli paketi aktif etmek için aşağıdaki kodu çalıştır:

```r
library(fredr)

```

* [ ] FRED API anahtarını tanıtmak için aşağıdaki kodu çalıştır:

```r
fredr_set_key("FRED_API_ANAHTARINI_BURAYA_YAZ")

```

* [ ] TCMB EVDS API anahtarını sisteme kaydetmek için aşağıdaki kodu çalıştır:

```r
Sys.setenv(EVDS_API_KEY="TCMB_API_ANAHTARINI_BURAYA_YAZ")

```

* [ ] Console'a aşağıdaki komutu yazıp Enter'a basarak EVDS anahtarının kaydedildiğini kontrol et:

```r
Sys.getenv("EVDS_API_KEY")

```
