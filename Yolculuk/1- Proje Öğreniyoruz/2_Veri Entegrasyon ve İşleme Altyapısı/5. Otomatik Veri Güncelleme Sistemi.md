```r
Sys.setenv(FRED_API_KEY = "KENDI_FRED_ANAHTARIN")
```

```r
library(fredr)
fredr_set_key(Sys.getenv("FRED_API_KEY"))
kur_verisi <- fredr(series_id = "DEXUSAL")
head(kur_verisi)

```

```r
veri_cek <- function() {
```

#### macOS (Darwin) kullanıyorsan bu bölümü uygula

* [ ] [macOS] Otomatik zamanlanmış görev için `cronR` paketini kur ve aktif et (`taskscheduleR` macOS'ta çalışmaz):

```r
install.packages("cronR")
library(cronR)

```

* [ ] [macOS] Script dosyasının gerçekten oluştuğunu doğrula (kaydettiğin tam yolu yaz; sonuç `TRUE` dönmeli):

```r
file.exists("/Users/KULLANICI_ADIN/proje/veri_guncelle.R")

```

* [ ] [macOS] Sonuç `TRUE` döndükten sonra script için cron komutu oluştur:

```r
cmd <- cron_rscript("/Users/KULLANICI_ADIN/proje/veri_guncelle.R")

```

* [ ] [macOS] Komutu her gün saat 09:00'da çalışacak şekilde zamanlanmış göreve ekle:

```r
cron_add(command = cmd, frequency = "daily", at = "09:00", id = "veri_guncelle")

```

* [ ] [macOS] Zamanlanmış görevin eklendiğini doğrula:

```r
cron_ls()

```

* [ ] [macOS] cron'un çalışabilmesi için Terminal'e Tam Disk Erişimi izni ver (Sistem Ayarları -> Gizlilik ve Güvenlik -> Tam Disk Erişimi); aksi halde görev sessizce çalışmaz

#### Windows kullanıyorsan bu bölümü uygula

* [ ] [Windows] Otomatik zamanlanmış görev için `taskscheduleR` paketini kur ve aktif et (`cronR` Windows'ta çalışmaz):

```r
install.packages("taskscheduleR")
library(taskscheduleR)

```

* [ ] [Windows] Script dosyasının gerçekten oluştuğunu doğrula (kaydettiğin tam yolu yaz; ters bölü işaretlerini çift yaz veya düz bölü kullan; sonuç `TRUE` dönmeli):

```r
file.exists("C:/Users/KULLANICI_ADIN/proje/veri_guncelle.R")

```

* [ ] [Windows] Sonuç `TRUE` döndükten sonra script'i her gün saat 09:00'da çalışacak şekilde zamanlanmış göreve ekle:

```r
taskscheduler_create(
  taskname = "veri_guncelle",
  rscript = "C:/Users/KULLANICI_ADIN/proje/veri_guncelle.R",
  schedule = "DAILY",
  starttime = "09:00"
)

```

* [ ] [Windows] Zamanlanmış görevin eklendiğini doğrula:

```r
taskscheduler_ls()

```
