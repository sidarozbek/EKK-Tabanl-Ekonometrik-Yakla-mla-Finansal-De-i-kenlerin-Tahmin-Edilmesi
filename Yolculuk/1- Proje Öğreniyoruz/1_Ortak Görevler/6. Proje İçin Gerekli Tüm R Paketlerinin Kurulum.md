* [ ] RStudio'yu aç
* [ ] File -> New File -> R Script yolunu izleyip boş bir kod sayfası aç
* [ ] Console ekranına aşağıdaki kodu yazıp çalıştır:

```r
install.packages(c(

"tidyverse",

"fredr",

"httr",

"jsonlite",

"OECD",

"wbstats"

))

```

* [ ] Kurulum sırasında Console'da soru çıkarsa varsayılan seçeneği kabul et veya `n` yazıp devam et
* [ ] Kurulum bitince aşağıdaki satırları kod sayfasına yaz:

```r
library(tidyverse)

library(fredr)

library(httr)

library(jsonlite)

library(OECD)

library(wbstats)

```

* [ ] Bu satırları seçip "Run" butonuna tıkla
* [ ] Console'da hata mesajı almadığını doğrula
* [ ] Dosyayı "veri_toplama_altyapisi.R" adıyla kaydet
