* [ ] RStudio'yu aç
* [ ] File -> New File -> R Script yolunu izleyip yeni bir kod sayfası aç
* [ ] Açılan sayfaya aşağıdaki kodu yazıp satırı seçtikten sonra "Run" butonuna basarak çalıştır:

```r
isim <- c("Ali", "Ayşe", "Efe")

```

* [ ] Alt satıra aşağıdaki kodu yazıp çalıştır:

```r
yas <- c(20, 22, 21)

```

* [ ] Alt satıra aşağıdaki kodu yazıp çalıştır:

```r
tablo <- data.frame(isim, yas)

```

* [ ] Sağ üstteki "Environment" sekmesinde tablo adında bir verinin oluştuğunu kontrol et
* [ ] Environment sekmesindeki tablo yazısının üzerine fareyle tıkla
* [ ] Sol üstte verinin bir Excel tablosu gibi yeni bir sekmede açıldığını doğrula
* [ ] Veri çerçevesindeki yaş sütununu görüntülemek için şu kodu çalıştır:

```r
tablo$yas

```

* [ ] Console ekranında yaş değerlerinin (20, 22, 21) göründüğünü doğrula
* [ ] Gerekli R paketlerini yüklemek için şu kodu Console'a gir ve çalıştır:

```r
install.packages(c("fredr", "shiny", "forecast"))

```

* [ ] Kurulum sırasında Console'da soru çıkarsa varsayılan seçeneği kabul et veya `n` yazıp devam et
* [ ] İndirme işleminin bitmesini bekle ve hata almadığını doğrula
