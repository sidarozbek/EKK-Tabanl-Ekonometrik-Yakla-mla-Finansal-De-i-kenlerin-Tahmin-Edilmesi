* [ ] RStudio'yu aç ve daha önce kaydettiğin kod dosyasını aç
* [ ] [FRED (Federal Reserve Economic Data)](https://fred.stlouisfed.org/) adresine gir, ücretsiz hesap oluştur ve API key al
* [ ] Gerekli paketi yüklemek için aşağıdaki komutu çalıştır:

```r
library(fredr)

```

* [ ] FRED API anahtarını aktif hale getirmek için aşağıdaki komutu çalıştır ve tırnak içine kendi anahtarını yapıştır:

```r
fredr_set_key("KENDI_API_ANAHTARINI_BURAYA_YAZ")

```

* [ ] Bir sonraki satırda şu komutu çalıştırarak veriyi al ve DEXUSAL kodunun istediğin döviz çiftine ait olduğunu kontrol et:

```r
kur_verisi <- fredr(series_id = "DEXUSAL")

```

* [ ] Sağ üstteki "Environment" sekmesinde "kur_verisi" adında yeni bir veri setinin oluştuğunu kontrol et
* [ ] Console ekranına aşağıdaki komutu yazıp Enter'a basarak verinin ilk 6 satırını görüntüle:

```r
head(kur_verisi)

```
