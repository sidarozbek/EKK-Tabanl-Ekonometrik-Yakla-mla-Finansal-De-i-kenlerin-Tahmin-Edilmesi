* [ ] RStudio'yu aç
* [ ] Sol üstten File -> New File -> R Script yolunu izleyip boş bir kod sayfası aç
* [ ] Açılan kod sayfasına aşağıdaki satırı yaz:

```r
x <- 10

```

* [ ] Alt satıra aşağıdaki kodu ekle:

```r
y <- 20

```

* [ ] Bir alt satıra aşağıdaki işlemi yaz:

```r
x + y

```

* [ ] Bu üç satırı fareyle seç ve üstteki "Run" butonuna tıkla
* [ ] Sol alttaki Console ekranında sonucun 30 olduğunu doğrula
* [ ] Sağ üstteki "Environment" sekmesinde x ve y değişkenlerinin oluştuğunu kontrol et
* [ ] Console ekranına aşağıdaki komutu yazıp Enter tuşuna bas ve kurulum tamamlanana kadar bekle

```r
install.packages("tidyverse")

```

* [ ] Kurulum tamamlandıktan sonra Console'a aşağıdaki komutu yazıp çalıştır ve hata mesajı almadığını kontrol et:

```r
library(tidyverse)

```

* [ ] Sol üstten File -> Save yolunu izleyerek kod dosyasını anlamlı bir isimle (örneğin `ilk_calisma.R`) bilgisayarına kaydet
