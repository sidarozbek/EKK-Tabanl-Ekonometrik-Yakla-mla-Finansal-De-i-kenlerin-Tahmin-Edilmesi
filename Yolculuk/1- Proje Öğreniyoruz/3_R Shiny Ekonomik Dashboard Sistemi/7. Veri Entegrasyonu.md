- Aşağıdaki fonksiyonu ekle

```r
veri_cek <- function(seri_kodu, baslangic = "2010-01-01") {

  fredr(
    series_id = seri_kodu,
    observation_start = as.Date(baslangic)
  ) %>%
    select(date, value) %>%
    drop_na()

}
```

- Console’a aşağıdaki kodu yaz ve çalıştır

```r
df <- veri_cek("UNRATE")tail(df, 3)
```
