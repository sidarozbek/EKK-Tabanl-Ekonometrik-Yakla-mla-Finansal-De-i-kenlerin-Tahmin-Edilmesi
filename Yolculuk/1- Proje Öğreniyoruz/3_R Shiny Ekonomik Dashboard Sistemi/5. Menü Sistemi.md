- [ ] Şu kısmı bul: **`dashboardSidebar()`**
- [ ] Aşağıdaki kodu **`dashboardSidebar()`** içine yaz

```r
dashboardSidebar(

  sidebarMenu(

    menuItem(
      "Yönetici Özeti",
      tabName = "yonetici",
      icon = icon("chart-line")
    ),

    menuItem(
      "Tahmin Laboratuvarı",
      tabName = "laboratuvar",
      icon = icon("flask")
    ),

    menuItem(
      "Model Röntgeni",
      tabName = "model",
      icon = icon("microscope")
    )

  )

),
```

- Şu kısmı bul: **`dashboardBody()`**
- Aşağıdaki kodu **`dashboardBody()`** içine yaz

```r
dashboardBody(

  tabItems(

    tabItem(
      tabName = "yonetici",
      h2("Yönetici Özeti")
    ),

    tabItem(
      tabName = "laboratuvar",
      h2("Tahmin Laboratuvarı")
    ),

    tabItem(
      tabName = "model",
      h2("Model Röntgeni")
    )

  )

)
```
